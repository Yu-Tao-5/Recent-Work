# Top AI Conference Affiliation Analysis

This repository contains a small case study for analyzing affiliation data in top AI conferences, with a focus on:

- company participation
- research organization involvement
- industry–academia collaboration
- country and region distribution

The project is based on paper metadata from:
- NeurIPS
- ICML
- CVPR
- ICLR
